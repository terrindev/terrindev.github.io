Roundly Privacy Policy

Effective Date: March 2026

1. Information We Collect

When you connect a financial account via Plaid, Roundly may receive transaction data, account identifiers, balances, and round-up amounts.

Roundly does not collect or store bank login credentials or full account numbers.

2. How We Use Information

We use collected data solely to calculate round-ups, display savings progress, and improve application functionality. We do not sell or rent user data.

3. Data Security

All data is transmitted via HTTPS/TLS encryption. Backend infrastructure is hosted in a secure cloud environment. API secrets are stored securely as environment variables and are not exposed client-side.

4. Data Retention

Roundly retains only the minimum data necessary to provide round-up tracking functionality. Users may request deletion of their data by contacting us.
